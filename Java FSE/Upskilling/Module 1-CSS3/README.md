# CSS3 Exercises - Local Community Event Portal

Open `index.html` in a browser to view the completed CSS3 exercise.

## Exercise Coverage

1. Inline, internal, and external CSS are shown in `index.html`.
2. `styles.css` uses formatted rules and descriptive comments.
3. Universal, element, ID, class, and grouping selectors are used.
4. HEX, RGBA, background image, fallback color, and gradients are used.
5. Google Font import and typography properties are applied.
6. Link pseudo-classes and list styling are included.
7. Event schedule table uses borders, padding, zebra striping, and collapse.
8. Event cards, selected fields, and visibility/display examples show box model behavior.
9. Community bulletin uses CSS3 multi-column layout.
10. Media query below 768px stacks navigation and adjusts layout/image/font sizing.
11. Use browser DevTools to test responsive screens and confirm `styles.css` loads in the Network tab.
