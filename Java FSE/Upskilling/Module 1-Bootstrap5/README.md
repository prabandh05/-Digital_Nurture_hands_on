# Bootstrap 5 Exercises - Local Community Event Portal

Open `index.html` to view the completed Bootstrap 5 CDN exercise page.

## Exercise Coverage

1. `index.html` links Bootstrap 5 through CDN. `local-bootstrap.html` shows downloaded/local file setup.
2. `vendor/bootstrap/README.md` explains CSS, JS, and icons folders. Both HTML pages include Bootstrap JavaScript bundle usage.
3. Responsive grid uses mobile, tablet, and desktop column classes.
4. Sidebar/content and four-column layouts use Bootstrap grid classes.
5. Alignment and reordering use `justify-content-center`, `align-items-center`, `order-md-1`, and `order-md-2`.
6. Flex utilities use `d-flex`, `flex-column`, `flex-md-row`, `justify-content-between`, and `align-items-center`.
7. Typography utilities include `display-1`, `lead`, `text-muted`, `fw-bold`, and text transform helpers.
8. Forms use `form-control`, `form-check`, `input-group`, and `form-floating`.
9. Buttons include contextual classes, outline buttons, button groups, and checkbox toggle buttons.
10. Navigation includes a responsive navbar, tabs, and pills.
11. Cards and media-style layouts are built with cards and flex utilities.
12. Spacing utilities appear in layout and pricing sections.
13. Contextual colors and `bg-gradient bg-dark text-white` are used.
14. Display utilities show and hide content by breakpoint, including a responsive sidebar.
15. Borders, shadows, rounded images, rounded pills, and rounded cards are included.
16. Fixed footer and image badge overlay use positioning utilities.
17. Bootstrap Icons appear in footer social links and icon-only buttons.
18. Modal and accordion demonstrate Bootstrap JavaScript plugins.
19. Sass customization scaffolding is in `scss/_variables.scss`, `scss/custom.scss`, and `package.json`.

## Optional npm Setup

Run these commands when internet access is available:

```bash
npm install
npm run build:css
```

To use downloaded Bootstrap files, copy Bootstrap and Bootstrap Icons distribution files into the `vendor` folders described in their README files, then open `local-bootstrap.html`.
